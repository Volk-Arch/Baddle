# baddle

> AHI Protocol — Augmented Human Intelligence.
> Не спрашивай у ИИ ответ — управляй процессом мышления.

**[English version](README_EN.md)** | **[Установка](SETUP.md)** | **[Видение](VISION.md)** | **[TODO](TODO.md)**

Граф мыслей с автономным циклом мышления. Задаёшь цель, выбираешь режим, нажимаешь Run — система думает сама: генерирует, сомневается, синтезирует. Результат — структурированный текст, проработанный с разных сторон. На любой LLM через OpenAI-совместимый API.

---

## Архитектура

Четыре слоя, каждый отвечает за своё:

```
┌─────────────────────────────────────────────────┐
│  РЕЖИМЫ (modes.py)                              │
│  13 конфигов: Free, Scout, Vector, Horizon...   │
│  Каждый = preset для Horizon + tick + stop       │
├─────────────────────────────────────────────────┤
│  HORIZON (horizon.py)               [планируется]│
│  Адаптивный контроллер: precision → temperature  │
│  policy_weights → выбор фазы                     │
│  surprise/novelty → обратная связь               │
├─────────────────────────────────────────────────┤
│  TICK (thinking.py)                              │
│  Цикл: generate → merge → elaborate → doubt      │
│  Фазы выбираются по состоянию графа              │
├─────────────────────────────────────────────────┤
│  ГРАФ (graph_logic.py)                           │
│  Ноды + рёбра + Байес + embeddings + кластеры    │
│  Smart DC · Pump · Collapse · Walk               │
└─────────────────────────────────────────────────┘
         ↕ API (api_backend.py)
   LM Studio / llama-server / OpenAI / ...
```

### Слой 1: Граф

Основа всего. Каждая мысль — нода с confidence (0–1). Связи — по embedding similarity. Evidence обновляет confidence через Bayes.

| Компонент | Что делает |
|-----------|-----------|
| **Байес** | P(H\|E) — уверенность узла обновляется при добавлении evidence |
| **Embeddings** | Cosine similarity между нодами → автоматические рёбра |
| **Кластеризация** | Connected components → группировка для merge |
| **Smart DC** | Тезис + антитезис + нейтраль → центроид → синтез с mathematically обоснованным confidence |
| **Pump** | Двусторонняя экспансия двух нод → LLM-абстракция → третий элемент (мост) `[планируется]` |

### Слой 2: Tick — цикл мышления

Автономный цикл, моделирующий как думает человек:

```
GENERATE    мало идей?            → Brainstorm (набрать массу)
MERGE       есть похожие?         → Collapse (слить ДО углубления)
ELABORATE   есть голые?           → Elaborate (добавить evidence)
DOUBT       есть непроверенные?   → Smart DC (усомниться)
GENERATE+   всё проверено?        → Brainstorm ("что упустил?")
SYNTHESIZE  ничего нового         → Финальный результат
```

Сходимость через три механизма:
- **Novelty check** — embedding similarity отсекает повторы
- **Lineage tracking** — `collapsed_from` не даёт перемалывать синтезы
- **Phase completion** — каждая фаза отрабатывает до конца

### Слой 3: CognitiveHorizon — адаптивный контроллер `[планируется]`

Управляет **как** генерировать, не **что**. Три параметра:

| Параметр | Суть | Влияние |
|----------|------|---------|
| **Precision (Π)** | Уверенность в модели (0–1) | → temperature: выше Π = ниже temp = фокус |
| **Policy (Λ)** | Веса фаз {generate, merge, elaborate, doubt} | → какую фазу выбрать на этом шаге |
| **Context (Γ)** | Активная рамка (промпт, правила) | → переключается при высокой novelty |

Обратная связь после каждого шага:
- **surprise** (1 - confidence после SmartDC) → корректирует precision
- **gradient** (+1 успех, -1 провал) → обновляет веса фаз
- **novelty** (1 - max similarity с историей) → триггер смены контекста

Четыре состояния: EXPLORATION (широко ищем) → EXECUTION (фокусируемся) → RECOVERY (что-то пошло не так) → INTEGRATION (синтезируем).

### Слой 4: Режимы — алгебра мышления

Любая задача = три вопроса:
1. **Сколько целей?** — 0 / 1 / 2+
2. **Какой примитив?** — `none / AND / OR / XOR`
3. **Какой тип цели?** — finite / repeatable / open

Из этого — **12 режимов** + свободный:

| Слой | Режим | Примитив | Стратегия | Пример |
|------|-------|----------|-----------|--------|
| — | **Свободный** | — | ручной | Все инструменты, без autorun |
| 0 целей | **Разведка** | none | — | Серфинг, блуждание |
| 1 цель | **Вектор** | focus | finite | Сдать проект |
| | **Ритм** | focus | repeatable | Зарядка каждый день |
| | **Горизонт** | focus | open | Изучение темы |
| AND | **Конструктор** | AND | unordered | Сборка мебели |
| | **Конвейер** | AND | SEQ | Рецепт, чек-лист |
| | **Каскад** | AND | PRIORITY | Тайм-менеджмент |
| | **Весы** | AND | BALANCE | Бюджет |
| OR | **Гонка** | OR | finite | Любое такси |
| | **Веер** | OR | open | Мозговой штурм |
| XOR | **Турнир** | XOR | comparative | Одна машина из трёх |
| | **Диспут** | XOR | dialectical | Научная дискуссия |

Каждый режим = preset для Horizon (precision, policy weights, target surprise) + tick-поведение + stop-условие.

> **Статус:** реализованы Свободный, Горизонт, Веер. Остальные — в [TODO](TODO.md).

### Стоп-условия (по типу цели)

| Тип | Стоп | Состояние после |
|-----|------|----------------|
| **finite** | `confidence ≥ threshold` | RESOLVED → архив |
| **repeatable** | `step_complete + trigger` | SCHEDULED → ждёт триггера |
| **open** | `diminishing_returns` | PARKED → ждёт новых данных |

## Операции с графом

| Операция | Что делает | Вход |
|----------|-----------|------|
| **Brainstorm** | N независимых идей по теме ноды | Нода |
| **Expand** | Новый угол от ноды (sibling) | Нода |
| **Elaborate** | Углубление с evidence (child) | Нода |
| **Ask** | Probing-вопрос к ноде | Нода |
| **Rephrase** | Переформулировка | Нода |
| **Verify (Smart DC)** | Тезис/антитезис/синтез → confidence | Нода |
| **Collapse (Merge)** | Синтез группы в один узел | Группа нод |
| **Walk** | Random Walk по transition probabilities | Нода |
| **Pump** | Поиск третьего элемента между двумя нодами `[план.]` | Две ноды |

## Интерфейс

Два режима: **Graph** (основной) и **Chat** (вспомогательный).

### Graph
- **Mode selector** — выбор режима, placeholder и tooltip меняются
- **Actions bar** — Select, Collapse, Link, Flow, Layout, Run, Undo, Save, Reset
- **SVG граф** — drag, zoom, pan, цветовая кодировка по типу и confidence
- **Detail panel** — Brainstorm, Expand, Elaborate, Ask, Rephrase, Verify, Walk, Evidence
- **Generation Studio** — пакетные варианты для любой операции
- **Run overlay** — Step / Phase / Hyp / Verified / Avg + sparkline сходимости

### Chat
Диалог с моделью. Context sidebar связывает граф ↔ чат.
