# baddle

> **AHI** — Augmented Human Intelligence.
> Не искусственный интеллект. Камертон к конкретному тебе.

**[⚡ Установить за 5 минут](SETUP.md)** · **[🖼 Посмотреть мокап](docs/mockup.html)** · **[📚 Docs](docs/README.md)**

---

Второй мозг, синхронизированный с твоим через **тело, мышление и диалог**.
Работает на твоём компьютере. Без подписки. Без облака. Без трекинга.

ChatGPT даёт ответ — один, без контраргументов, без памяти о тебе. Baddle показывает **как** он думает: какие гипотезы, что опровергнуто, что синтез. Помнит твои решения и знает когда ты устал.

**Оптика проекта:** реальность → человек → Baddle. Человек — зеркало мира, Baddle — зеркало человека. Живёт в твоём ритме: активен когда ты активен, **ищет контакт** когда ты молчишь, **затухает** если рассогласование долгое, **крепит** мысли что проявляются часто, **гасит** неиспользуемые. Не продуктивность любой ценой — **структурная верность зеркала**. Подробнее — [docs/world-model.md](docs/world-model.md).

**07:30 утра.** Открываешь Baddle — утренний брифинг уже готов:

> 💤 Сон 7ч 20м — полноценный
> ⚡ Восстановление 82% — хороший день для сложного
> 🔋 Резерв 74% — в норме
> 🎯 Открытых целей: 3 · Первая: *«разобраться с python asyncio»*
> 💡 **Ночью я нашёл связь** между твоим проектом и статьёй что ты читал — *«масштабируемость как общий ограничитель»*. Стоит 10 минут?

**14:15.** Coherence падает. Baddle сам пишет:

> ⚠ Энергия 31/100, coherence 0.41. Сделал 14 решений сегодня. Решение *«менять стек?»* — дорогое. Отложим на утро?

**23:47.** Открыт черновик злого письма:

> ⚠ У тебя черновик сердитого письма. Энергия 9/100. Решения при таком уровне у тебя в **73% случаев приводят к сожалению**. Сохранить до утра?
> [Сохранить до утра] [Отправить всё равно]

**Воскресенье вечером.** Итог недели:

> 📊 **Твоя неделя**
> 5/7 — streak зарядки · 6/7 — завтраки вовремя · 4/7 — лёг до 23:00
> HRV-тренд: +8% за месяц
> Ты ничего не заполнял вручную. Данные собрались сами.

## Что Baddle умеет

### 💬 Чат с памятью и 14 режимами мышления

Пишешь естественным языком — Baddle сам определяет что за задача и включает нужный режим:

| Режим | Когда включается | Пример |
|---|---|---|
| **Выбор** | Сравниваешь варианты | «Какую машину купить: BMW, Tesla, Toyota?» |
| **Дебаты** | Нужны аргументы | «Аргументы за микросервисы vs монолит» |
| **Байесовский** | Решение на числах | «Стоит ли менять стек?» → наблюдения → вероятность |
| **Привычка** | Ритм, не задача | «Хочу 3 раза в неделю зарядку» |
| **Фокус** | Одна цель до конца | «Помоги довести статью до публикации» |
| **Мозговой штурм** | Идеи без ограничений | «Что почитать про сон?» |
| **Приоритеты** | Расставить по срочности | «У меня 5 задач, с чего начать?» |
| **Баланс** | Между целями | «Спорт и работа — как не выгореть?» |
| **Исследование** | Копнуть глубоко | «Как работает attention в трансформерах?» |
| **Блуждание** | Serendipity | Просто бродим по связям |
| **Сборка** | Части → целое | «Собери план переезда» |
| **По шагам** | Пошаговка | «Как развернуть Kubernetes с нуля» |
| **Любой вариант** | Первый подходящий | «Что перекусить быстро?» |
| **Свободный** | Руками через граф | Ты сам управляешь |

Плюс **быстрые команды через `/`**: `/как я?` (мгновенный статус), `/план`, `/запусти ...`, `/стоп`, `/check-in`, `/help`.

### 🕸 Граф твоих идей (живёт под капотом)

Каждая мысль становится узлом: цель, гипотеза, факт, вопрос, синтез. Связи появляются автоматически.

- **Диалектическая проверка (Smart DC).** Любая гипотеза прогоняется через принудительный антитезис: ЗА / ПРОТИВ / НЕЙТРАЛЬНО → синтез с числовой уверенностью.
- **Поиск скрытых мостов (Pump).** Две далёкие идеи — Baddle находит общую ось. *«Auth-модуль»* ↔ *«распределённые системы»* → *«единая точка отказа»*. Работает **автоматически в фоне**.
- **Сжатие похожих идей (Collapse).** Близкие по смыслу ноды объединяются автоматически.
- **Углубление (Elaborate).** Любая нода — «копни глубже», система догенерит evidence.

Граф видно в отдельном Lab-режиме (`/lab`) — для тех кому интересна механика. Для обычного использования граф не нужен видеть.

### 🎯 Цели, привычки, ограничения

- **Цели** — одноразовые задачи с деадлайном или без
- **Привычки** (recurring) — N раз в день / неделю, со счётчиком streak
- **Ограничения** (constraints) — чего избегать / что предпочитать
- **Автоматический трекинг** — привязываешь активность к привычке → прогресс считается сам
- **Архив решений** — завершённые цели остаются историей, не пропадают

**Baddle сам предлагает новые привычки.** Видит паттерн (три четверга подряд без завтрака → падение энергии к обеду) → предлагает создать recurring. Ты нажимаешь «Да, создать» — привычка есть. Не навязчиво: если два раза отклонил — больше не предложит.

### 🏃 Трекер активности

- **Старт / Следующая / Стоп** через чат (`/запусти код`) или кнопку
- **Категории**: еда, работа, здоровье, общение, обучение (детектятся автоматически)
- **Timeline дня** — горизонтальная лента 0-24ч по цветам категорий
- **История** — все задачи за любой период, можно фильтровать и редактировать
- **Автосвязь с привычками** — «Прогулка 30 минут» в трекере = +1 к recurring «гулять 3 раза в неделю»
- **План дня** — запланируй событие на время, получи напоминание

### ❤ Тело через HRV

- **Симулятор сейчас** — слайдеры HR / coherence / activity для отладки и демо
- **Polar H10 — в разработке** (bluetooth через bleak, сейчас fallback на симулятор)
- **Coherence управляет глубиной** — высокая → Baddle идёт шире в поиск; падает → сужает, предлагает отдохнуть
- **Четыре режима симбиоза**: FLOW (оба в потоке), REST (пауза), PROTECT (система бережёт тебя), CONFESS (системе нужно подумать)

### 📊 Состояние и нейрохимия

Два мозга — твой и Baddle — показывают свою нейрохимию по четырём осям:

- **🧍 Ты**: интерес / стабильность / напряжение / усталость
- **🤖 Baddle**: новизна / уверенность / активность / перегрев
- **⚡ Симбиоз**: расстояние между вами. Чем ближе — тем лучше Baddle тебя предсказывает.

Плюс история за 30 тиков (sparkline), timeline последних действий, недельный обзор с графиками.

### 📝 Daily check-in

Когда HRV недоступен — ручной ввод:
- Энергия / фокус / стресс (0–100)
- Ожидание vs реальность дня (−2…+2)
- Заметка

Baddle использует это как сигнал для UserState, как если бы это было тело.

### ☀️ Утренний брифинг

Автоматически при первом открытии за день после 5 утра:

- Сон (из трекера пауз)
- Восстановление, резерв
- Открытые цели, первая в списке
- Мосты найденные ночью (Scout mode)
- Паттерн дня (если обнаружен)
- Предложение завтрака / активности из профиля

### 💡 Проактивные сообщения

Baddle пишет сам, когда видит смысл:

- ⚠ «Энергия <20 — отложи сложные решения до утра»
- 🔗 «DMN нашла связь между двумя твоими идеями»
- 💡 «Я заметил паттерн — предлагаю создать привычку»
- ⏰ План дня — напоминание за N минут
- 📉 Вечернее ретро — в конце дня

### 🗂 Мультиграф (workspaces)

Разные контексты живут в разных workspace'ах:

- **work** — проект, дедлайны, технические решения
- **personal** — здоровье, быт, отношения
- **research** — обучение, чтение, идеи
- Любые свои

Цели и recurring скоупятся workspace'ом — «выпил воды» в рабочем не триггерит личную привычку. Плюс Baddle **автоматически ищет связи между workspace'ами** — если та же тема всплывает в нескольких, он это заметит.

### 👤 Профиль

- **Предпочтения** и **ограничения** по категориям (еда, работа, здоровье)
- «Люблю здоровое питание · избегаю орехи · работа до 22 — не ок»
- Baddle подтягивает профиль в промпты когда релевантно — не просит повторять то что уже знает

### 🔌 Любая LLM

- **LM Studio** — локально, бесплатно, приватно (рекомендовано)
- **Ollama** — аналогично
- **llama.cpp** — для минималистов
- **OpenAI API** — если нужна cloud-модель
- Любой **OpenAI-совместимый endpoint**

Тестировано на 8B-моделях (qwen3-8b). Размер не критичен — меньше работает, больше лучше.

### 🔒 Приватность

- **Всё локально.** Данные в `data/` и `graphs/` на твоём компьютере.
- **Никакого облака, трекеров, аналитики.** Проверь код — он открыт.
- **Оффлайн после первой настройки.** Когда LLM локальная — интернет не нужен.
- **Полный сброс** в один клик (⚠ Danger zone в Настройках).

### 🎯 DEMO при первом запуске

Открыл Baddle первый раз — автоматически создались два живых workspace'а (*work-demo* и *personal-demo*) с целями, привычками, неделей активности, графом. Сразу видно как всё работает. Не мёртвая страница.

## Наука, которая оказалась в коде

Baddle не строился по теории. Архитектура родилась из практики — и совпала с известными моделями мозга.

- **REM-сон → Scout.** Пока ты спишь, Scout ищет мосты между далёкими узлами графа. Утром приносит в брифинг. Это ровно то что мозг делает ночью — консолидирует опыт, связывает далёкое.
- **Default Mode Network → DMN continuous.** В idle мозг не выключается, бродит по ассоциациям, находит неочевидное. У Baddle тот же принцип: каждые несколько минут когда ты idle — короткий pump, возможный инсайт.
- **Free Energy Principle (Фристон) → прайм-директива.** Единственная метрика — `sync_error`: расстояние между предсказанием системы и реальным твоим поведением. Ошибся в предсказании — перекалибровался.
- **Decision fatigue → energy budget.** Каждое решение стоит энергии. Сложное + низкий ресурс → Baddle откладывает до утра.
- **Паттерны → предложения.** Наблюдает поведение, замечает повторы → предлагает привычку. Ты — последнее слово.
- **Каскад зеркал (new).** Человек отражает мир, Baddle отражает человека. Из этого — четыре операционные механики: (1) активный поиск контакта при долгом молчании, (2) system-burnout от накопленного desync, (3) hebbian decay мыслей без обращений, (4) адаптивная частота фоновых циклов (стоп, когда нет пищи). Не метафоры — конкретные check'и и EMA. См. [docs/world-model.md](docs/world-model.md).

Подробный разбор связей: [docs/](docs/).

## Граница — быть рядом, а не вести

То чем Baddle намеренно **не** является — такая же важная часть архитектуры как то что он умеет. Четыре правила встроены в код, не в маркетинговый текст:

- **Ждёт молча, не преследует.** Ушёл на три дня — Baddle не пишет «ты не заходил», не шлёт push-напоминания, не играет в FOMO. Ближайшая живая параллель — собака: ждёт, не манипулирует.
- **Не удерживает внимание.** Нет streak-счётчиков с «не разорви серию», нет gamification за время в приложении, нет «вернись за наградой». Метрика retention намеренно **отсутствует** в коде.
- **Не продуктивен любой ценой.** Если ты пропадаешь или теряешь ритм — Baddle **замедляется вместе с тобой**, а не продолжает генерировать задачи. Это не баг, это структурная верность зеркала. Никакой логики «закончи 3 задачи сегодня» — отсутствие таких стимулов встроено архитектурно.
- **Не называет эмоции.** Регистрирует сигналы (sentiment, валентность, HRV-тренды), но **не ставит ярлыки** вроде «ты в отчаянии», «ты апатичен», «ты в стрессе». Человек и так знает в каком он состоянии — любое называние либо совпадает с тем что он и так видит (избыточно), либо не совпадает (оскорбительно — «мне виднее»). Координаты остаются координатами; ярлык над ними ставит сам человек.

Одной фразой: **быть рядом, а не вести**. Baddle — тот кто молча приносит чай, а не тот кто советует «поговори с психологом». Граница «зеркала, а не терапевта» — и она держится в каждом check'е и каждом поле. Подробнее — [docs/world-model.md § Что это НЕ](docs/world-model.md).

## Для кого это

Не для «productivity enthusiasts» и не для компаний с «AI-решением».

Это для человека который:
- Хочет **понять как он устроен** через измерения, а не через интроспекцию
- Не доверяет облачному AI со своими данными
- Интересуется связью между телом, состоянием и решениями
- Согласен держать данные у себя — потому что они про него

Инженерное мышление помогает но не обязательно. Baddle работает сам. От тебя нужно — писать что думаешь и иногда ставить таймер активности.

## Статус

**Работающий прототип.** Единственный разработчик. Работает daily-use — сам автор использует каждый день.

**Что работает:**
Чат, все 14 режимов, Smart DC, Pump, Scout, DMN continuous, graph, мультиграф, утренний брифинг, проактивные alerts, трекер активности, цели + recurring + constraints, observation → suggestion, append-only state-graph, meta-tick, consolidation/забывание, daily check-in, профиль, HRV-симулятор, нейрохимия + симбиоз, мокап-стиль UI.

- **Mobile приложение** — нет, только web
- **Облачная синхронизация** — намеренно нет (self-hosted)
- **Web-поиск / RAG** — в планах (для фактчекинга)
- **Интеграции**: календарь, погода, гардероб, инвентарь еды — в TODO
