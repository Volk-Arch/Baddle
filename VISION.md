# Baddle: AHI Protocol

**Augmented Human Intelligence** — усиленный человеческий интеллект.

---

## Основной режим: из A в B

Всё сводится к навигации. Выбираешь точку A (где ты сейчас) и точку B (куда хочешь прийти). Система автоматически строит путь и ведёт по нему.

| Задача | Точка A | Точка B | Как система ведёт |
|---|---|---|---|
| Здоровье | "пропускаю обед" | "3 приёма в день" | watchdog + напоминания в нужное время |
| Статья | "5 разрозненных мыслей" | "структурированный текст" | Smart DC + коллапс по пути |
| Решение | "паралич выбора" | "решение принято" | tie_breaker + марковские последствия |
| Обучение | "не знаю тему" | "confidence > 0.8" | elaborate по слабым узлам |

Transition_prob показывает оптимальный путь. tick() двигает по нему. Человек задаёт A и B — система ведёт, но каждый шаг требует merge от человека.

---

## Что это

Граф мыслей с тремя движками:

| Движок | Что делает | Пример |
|---|---|---|
| **Байес** | Уверенность узла 0.0–1.0 | "Удалёнка эффективнее" → conf=0.5, после проверки → 0.82 |
| **Марков** | Вероятность перехода между узлами | A→B: 0.7, A→C: 0.3. "Куда ведёт мысль?" |
| **Smart DC** | Тезис + антитезис + нейтраль → синтез | Слабая гипотеза → три полюса → центроид → проверенный узел |

---

## Как работает

```
Ввод мысли → confidence = 0.5 (гипотеза)
       ▼
Оценка → conf < 0.8 → "горячая точка"
       ▼
Smart DC: тезис / антитезис / нейтраль → центроид → синтез (conf = 0.82)
       ▼
Рекурсия → conf > 0.8 → стоп (иначе ещё цикл)
       ▼
Человек: ✓ Merge | ✗ Reject | ✎ Edit | ↻ Retry
```

**Откуда числа:**
- Confidence: logprobs при генерации → расстояние синтеза до центроида → ручная корректировка
- Transition_prob: cosine similarity → directed edges → навигация пользователя (чаще кликаешь — вес растёт)

## Принципы

**Человек — Root Admin.** ИИ предлагает, человек решает. Никаких автоматических merge.

**Энтропия — топливо.** Система ищет слабые места (conf < 0.4), не ждёт вопросов.

**Время — связь.** Мысли одной сессии связаны контекстуально, даже если семантически далеки.

**Граф — явная внешняя память.**

| В голове | В графе |
|---|---|
| Интуиция | `confidence: 0.73` |
| Ассоциации | Рёбра + `transition_prob` |
| Сомнения | Низкий `evidence_count` |
| "За и против" | Smart DC → синтез |
| История | Git log |

## Возможности

**Автономное блуждание.** ИИ обходит граф в фоне: находит слабые узлы, предлагает синтезы, ищет мосты между кластерами. Каждый шаг — коммит. Merge — только за человеком.

**Цикл день/ночь.** Днём — сбор мыслей (дивергенция). Ночью — автоколлапс по горячим точкам (конвергенция). Утром — структурированные выводы вместо хаоса.

**Ассоциативный доступ.** Не поиск по ключевым словам, а распространение активации по графу. Прямое (по смыслу), контекстное (по состоянию), непрямое ("12 раз пропустил обед" — паттерн).

**Cross-graph.** Несколько графов (Работа, Здоровье, Идеи) + связи между ними. Блуждание с перескоком: "недосып коррелирует с пиком креативных идей".

## Интеграция с LLM

| | AI без графа внутри | AI с графом внутри |
|---|---|---|
| Режим | **Буст** — даём память | **Симбиоз** — два графа соединяются |
| Результат | ИИ полезнее | Когнитивный партнёр |

Три слоя: LLM (смысл) + Байес (уверенность) + Марков (динамика).

---

## AGI vs AHI

| AGI | AHI |
|---|---|
| Заменяет человека | Усиливает человека |
| Чёрный ящик | Прозрачный граф |
| "Решает за вас" | "Предлагает, решаете вы" |
| Alignment через обучение (RLHF) | Alignment через архитектуру |

Безопасность по конструкции: граф виден, человек мержит, Байес не даёт быть догматичным, принудительный антитезис ломает эхо-камеры.

## Экосистема

**Baddle Desktop** — локальный ИИ + персональный граф. 100% приватность, zero cloud.

**Graph Store** — маркетплейс графов знаний (МБ, не ГБ). Доменные графы: медицина, право, наука.

**Git Verify** — MR для знаний, review экспертами, рейтинги. Научный метод в архитектуре.

### Применение

- **Образование** — граф знаний с confidence вместо оценок. Экзамен = merge request
- **Медицина** — симптомы → паттерны → Smart DC по слабому диагнозу → врач решает
- **Наука** — публикация = граф. Рецензия = MR. Fork = альтернативная интерпретация
- **Работа** — мессенджер без поиска → всё важное в граф → ночной коллапс → утром выводы
