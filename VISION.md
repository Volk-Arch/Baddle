# Baddle: AHI Protocol

**Augmented Human Intelligence** — усиленный человеческий интеллект.

Не искусственный интеллект, который заменяет. Усиленный человеческий, который делает мышление прозрачным, проверяемым и направленным.

---

## Научные основы

Baddle не реализует эти теории математически строго — он использует **ту же интуицию** в инженерном контексте. Precision → temperature — это аналогия, не изоморфизм. Но аналогия рабочая: каждый компонент решает ту же проблему что и его нейробиологический прототип.

### Active Inference и Free Energy Principle

*Карл Фристон: мозг минимизирует surprise — расхождение между ожиданием и реальностью.*

**В Baddle: CognitiveHorizon.** Precision (Π) контролирует баланс между prior beliefs и новыми данными. Высокий surprise → precision падает → система расширяет поиск гипотез (повышает temperature). Низкий surprise → precision растёт → фокусируется на проверенном (понижает temperature).

```
P_new = P_old + α * (surprise - target_surprise)
temperature = 1.0 - precision
```

Это не метафора. Это буквально predictive processing: система поддерживает модель мира (граф), сравнивает ожидания с результатами (surprise = 1 - confidence после SmartDC), и адаптирует параметры восприятия (precision → temperature/top_p).

### Байесовский мозг

*Мозг не хранит факты — он хранит распределения вероятностей над гипотезами.*

**В Baddle: confidence каждой ноды.** Нет "истинных" и "ложных" мыслей. Каждая — гипотеза с confidence от 0 до 1. Evidence (supports/contradicts) обновляет через Bayes:

```
P(H|E) = P(E|H) × P(H) / P(E)
```

Alpha/Beta модель отслеживает баланс поддерживающих и противоречащих свидетельств. Модель не может быть догматичной — confidence никогда не равна 1.0.

### Диалектика (Гегель)

*Тезис → антитезис → синтез. Истина рождается из столкновения противоположностей.*

**В Baddle: Smart DC.** Три параллельных генерации (тезис/антитезис/нейтраль) → три вектора в embedding-пространстве → центроид (математическая точка равновесия) → синтез ближайший к центроиду. Confidence = cosine similarity синтеза к центроиду.

Принудительный антитезис — ключевое. Модель не может просто согласиться. Архитектура обязывает показать другую сторону. Эхо-камеры ломаются структурно, не волевым усилием.

### Хеббовское обучение

*"Neurons that fire together wire together." Связи, которые часто активируются, усиливаются.*

**В Baddle: transition probabilities.** При навигации пользователя по графу `tp += lr × (1 - tp)` — часто используемые пути усиливаются. Граф учится от поведения пользователя: какие связи важны, какие — нет.

### Дивергентное / конвергентное мышление (Гилфорд)

*Два режима: расширение (генерация идей) и сужение (синтез, выбор).*

**В Baddle: цикл Generate → Merge → Doubt.** Brainstorm — дивергенция (набрать массу разных идей). Merge + Collapse — конвергенция (слить похожие, синтезировать). Цикл повторяется: каждое сужение поднимает уровень абстракции, каждое расширение ищет пробелы.

CognitiveHorizon формализует это: EXPLORATION state (precision↓, divergent) → EXECUTION state (precision↑, convergent). Переход автоматический, по обратной связи.

### Ограничения рабочей памяти (Миллер, 7±2)

*Человек удерживает ~7 объектов одновременно.*

**В Baddle: Merge/Collapse.** Граф может содержать 50 нод, но финальный синтез сжимает их в 4-6 проверенных идей. Каждый collapse = переход на уровень абстракции выше. Как человек: не помнишь 50 деталей, помнишь 5 выводов.

### Метапознание

*Мышление о мышлении. "Что я упустил?" "Где я застрял?"*

**В Baddle: META фаза + Run overlay.** После проверки всего — цикл спрашивает "что ещё?", видя список уже верифицированного. Оверлей показывает precision, width, фазу, sparkline — мышление наблюдает за собой.

### Хабитуация / детекция новизны

*Мозг перестаёт реагировать на повторяющиеся стимулы. Внимание привлекает только новое.*

**В Baddle: novelty check.** Embedding similarity > threshold → мысль отклоняется. Когда модель начинает повторяться — все новые мысли отклоняются → естественный стоп. Цикл останавливается не по лимиту, а потому что нейросеть исчерпала тему. Точно как мозг: "я уже это думал".

### Когнитивная гибкость

*Способность переключаться между режимами мышления в зависимости от задачи.*

**В Baddle: 13 режимов из 4 примитивов.** Не один способ думать, а алгебра: AND/OR/XOR + finite/repeatable/open = 12 паттернов + свободный. Каждый режим = свой preset CognitiveHorizon (precision, policy weights, target surprise). Переключение — смена пресета, не кода.

### Dual Process Theory (Канеман)

*System 1 — быстрое, интуитивное. System 2 — медленное, осознанное.*

**В Baddle: весь инструмент — это System 2, вынесенная наружу.** LLM в чат-режиме = System 1 (быстрый ответ, одним проходом, без проверки). LLM в Baddle = System 2 (граф, evidence, диалектика, перепроверка, синтез). Та же модель, но процесс другой.

### Семантические сети (Куиллиан)

*Знание как граф связанных понятий. Извлечение через spreading activation.*

**В Baddle: буквально.** Ноды = понятия. Рёбра = семантическая близость (cosine similarity embeddings). Walk = spreading activation. Pump (Накачка) = двусторонняя активация двух понятий до пересечения.

## Как концепции собираются в архитектуру

```
Научная концепция              → Компонент Baddle
─────────────────────────────────────────────────────
Active Inference / Precision   → CognitiveHorizon (Π, Λ, Γ)
Байесовский мозг               → confidence + Bayes update
Диалектика Гегеля              → Smart DC (thesis/antithesis/centroid)
Хеббовское обучение            → transition_prob += lr × (1 - tp)
Дивергенция / конвергенция     → Generate ↔ Merge/Collapse
Рабочая память (7±2)           → Collapse в 4-6 синтезов
Метапознание                   → META фаза + Run overlay
Хабитуация                     → Novelty check (embedding sim)
Когнитивная гибкость           → 13 режимов из 4 примитивов
System 2 (Канеман)             → Весь инструмент
Семантические сети             → Граф + Walk + Pump
```

## Два примитива операций

| | Smart DC | Pump (Накачка) |
|---|----------|----------------|
| **Научная основа** | Диалектика Гегеля | Spreading activation + абстракция |
| **Вход** | 1 нода | 2 ноды |
| **Метод** | Тезис/антитезис/синтез → центроид | Двусторонняя экспансия → LLM-абстракция |
| **Результат** | Confidence одной идеи | Третий элемент — мост между двумя |
| **Вопрос** | "Это правда?" | "Что связывает?" |

## Алгебра мышления

| Примитив | Суть | Стратегии |
|----------|------|-----------|
| **none** | Нет целей | — (Разведка) |
| **AND** | Все обязательны | unordered / SEQ / PRIORITY / BALANCE |
| **OR** | Любая достаточна | finite (Гонка) / open (Веер) |
| **XOR** | Ровно одна | comparative (Турнир) / dialectical (Диспут) |

Стоп по типу цели: finite → RESOLVED, repeatable → SCHEDULED, open → PARKED.
